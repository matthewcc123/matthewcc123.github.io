ON THE ROAD! (2021 WIP)

On The Road! it's a remake version of On The Road (Unity3D). There is a lack of fun on the last version because of the realistic physics. I decided to remake the game to bring back the original gameplay physics and also upgrade the graphic with optimization so the game can run smoother on low-end devices.

New Features :

- Day/Night Cycle
- Car Modifications
- Power-Ups
- Achievements
- Photo Mode
